FACTORIAL CALCULATOR

A factorial operation takes the product of all positive integers less than or equal to a number (n), denoted as n!. Example:

4! = 4 x 3 x 2 1 = 24

The design for this calculator was inspired by the NES game "Final Fantasy"

INPUT RULES

- Must be a positive integer

- Cannot be greater than 170 otherwise will return "Infinity"

- Decimals will be rounded to the nearest integer
